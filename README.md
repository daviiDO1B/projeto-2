# EcoPragas - Controle Ecológico de Pragas

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

#agrinho2026

## Objetivo

Apresentar soluções ecológicas para o controle de pragas agrícolas, incentivando práticas sustentáveis que preservam o meio ambiente e promovem uma produção agrícola mais equilibrada.

## Justificativa

O uso excessivo de defensivos químicos pode causar impactos ambientais e prejudicar a biodiversidade. Este projeto busca demonstrar alternativas sustentáveis que auxiliam no controle de pragas de forma eficiente e responsável.

## LINGUAGENS UTILIZADAS

* HTML5
* CSS3


## Funcionalidades

* Página responsiva.
* Botão de modo escuro.
* Efeitos visuais com hover.
* Manipulação do DOM via JavaScript.
* Interface adaptada para dispositivos móveis.

O JavaScript manipula elementos do DOM permitindo alternar entre modo claro e escuro. O CSS utiliza classes, IDs, Flexbox e Media Queries para garantir responsividade.
